# jsdoc-markdown-cli

```
Usage: jsdoc-markdown-cli example/**/*.js

Generates readme.md

Arguments:
  patterns          Glob patterns to match files

Options:
  -V, --version     output the version number
  -d --dry-run      Don't create files
  -c --clear-empty  Delete empty files
  -h, --help        display help for command
```
